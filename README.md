# Online Purchase Intention Prediction

Machine learning project focused on predicting whether an online shopping session will result in a purchase.

## Project overview

The project uses the Online Shoppers Purchasing Intention Dataset from the UCI Machine Learning Repository. It covers data quality assessment, exploratory data analysis, preprocessing, model training, validation and interpretation.

## Models

- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors

## Evaluation

The models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
- Cross-validation
- ROC-AUC
- Bootstrap validation

Hyperparameter tuning was performed using GridSearchCV. Random Forest achieved the best overall performance.
